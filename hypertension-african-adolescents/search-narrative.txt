This file reports the search methods for the review using the PRISMA extension for searching (Rethlefsen 2021).

Information Sources and Methods
MEDLINE (Ovid), Embase (Ovid), Web of Science Core Collection (Clarivate), and African Index Medicus (World Health Organization) were searched. In addition, the first 200 results from Google Scholar were exported using Harzing's Publish or Perish software. 

Search strategies
The strategy consists of three concepts, combined with Boolean operators: adolescents; African countries; and arterial hypertension. Both controlled vocabulary and free-text keywords were identified using the Yale MeSH Analyzer; proximity operators were incorporated in order to retrieve variations of phrases.
